
# 📊 Retail Sales Analytics with SQL (PostgreSQL)


## Overview

This project demonstrates an end-to-end retail sales analysis using PostgreSQL. The analysis focuses on solving real-world business problems by transforming raw sales data into actionable insights through SQL.

The project covers data cleaning, KPI calculation, customer analytics, sales trends, product performance, fulfillment analysis, and advanced SQL techniques such as CTEs, window functions, ranking, and rolling calculations.

---

## Objectives

- Clean and prepare retail sales data
- Analyze overall sales performance
- Measure customer purchasing behavior
- Identify top-performing products and categories
- Evaluate order fulfillment performance
- Generate business KPIs
- Practice advanced SQL concepts used in industry
